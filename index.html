Proejct:

SleepHunter

SleepHunter is a mobile service on mining sleep stages during sleep. It detects sleep stage by recording the user's physical activities and sleep environment. The aim of this project lays two fold. In the first, SleepHunter provides the sleep quality to users by the distribution of different sleep stages. Secondly, users can be woken up when SleepHunter detects the user falls in light sleep, which is good for their health.

Bubbles

Bubbles is a file management mobile service. Users can leverage it to manage files and share files with other users. Moreover, Bubbles can help users to transfer files by creating local network . Four versions of Bubbles have been developed on OS X, iOS, Windows and Android platforms.
